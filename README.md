📌 Project Overview

This project analyzes manufacturing defects in PVC pipe production using Statistical Quality Control techniques to identify major defect categories and evaluate process stability.

🎯 Objective

Identify dominant defect types

Evaluate process stability using control charts

Analyze economic impact of defects

Propose a preventive quality monitoring framework

📈 Methodology Used

Pareto Analysis

NP-Control Chart

Cause & Effect (Ishikawa) Diagram

Trend Analysis

Cost Impact Analysis

🔍 Key Findings

Crack and melt defects contributed to nearly 80% of total defects.

The production process was statistically stable but showed batch-wise variation.

Targeted control of major defect causes can significantly reduce rejection levels and production cost.
